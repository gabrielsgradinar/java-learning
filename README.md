# java-learning

# Compilar código Java
```shell

javac Hello.java # gera o arquivo .class com o código compilado
java Hello # executa o código compilado

```

# Objetos mútaveis e imutáveis

- tipos primitivos e Strings são objetos imutáveis


### ArrayList 
- usa um array normal por baixo dos panos
- tem a capacidade de alterar o tamanho dinamicamente

### LinkedList
- mais lento ao retornar os elementos
- mais rápido para remover/adicionar
